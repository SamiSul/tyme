# tyme

a simple vscode extension to help you track the time you spend on your tasks.

## commands

| command   | description   |
| --------- | ------------- |
| tyme.show | tyme: show ui |

## shortcuts

| shortcut        | description |
| --------------- | ----------- |
| alt + backspace | remove all  |
| alt + p         | pause all   |
| alt + r         | resume all  |
| alt + s         | stop all    |
| alt + enter     | add/confirm |
